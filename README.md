# 📘 Proyecto Actividades JavaScript

## 📌 Descripción

Este proyecto reúne una serie de actividades prácticas desarrolladas en JavaScript como parte del proceso de aprendizaje. Cada actividad está organizada de forma modular dentro de la carpeta `src/features`, facilitando la comprensión, mantenimiento y escalabilidad del código.

El objetivo principal es aplicar conceptos fundamentales de JavaScript mediante ejemplos claros, funcionales y fáciles de entender.

---

## 🚀 Deploy

La aplicación se encuentra desplegada correctamente y permite visualizar las diferentes actividades desarrolladas.

---

## 🧩 Actividades

### 1. 🗂️ CRUD con LocalStorage

Permite crear, leer, actualizar y eliminar registros almacenados en el navegador utilizando LocalStorage.

### 2. 🔄 Callback

Ejemplo práctico del uso de funciones callback para controlar la ejecución de funciones.

### 3. ⏳ Promesas (Promises)

Implementación de promesas para el manejo de operaciones asíncronas de manera eficiente.

### 4. ⚡ Async/Await

Uso de `async` y `await` como una forma moderna y más legible de trabajar con promesas.

### 5. 📦 Destructuring

Aplicación de destructuring para extraer propiedades de objetos de manera más limpia y sencilla.
